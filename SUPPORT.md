# Support

## Documentation

Start with the public website and documentation:

- [pgkinetic.dev](https://pgkinetic.dev)
- [docs.pgkinetic.dev](https://docs.pgkinetic.dev)

The core proxy, container, Helm chart, and operational documentation are
maintained in [HookWoods/pg-kinetic](https://github.com/HookWoods/pg-kinetic).

## Website Issues

For a reproducible website problem, open a GitHub issue and include:

- affected URL and deployment URL when applicable
- browser, operating system, device, and viewport dimensions
- expected and observed behavior
- minimal reproduction steps
- a screenshot or safe console output when useful

Search existing issues first. Keep one issue focused on one user-facing problem.

## Security

Do not report vulnerabilities, API keys, Cloudflare secrets, or analytics
credentials in a public issue. Follow [SECURITY.md](SECURITY.md) for private
reporting.
